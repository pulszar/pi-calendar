# pi-calendar

## Intention

Google Calendar API powered dashboard purpose-built for a small display.

## Future Goal

Add more specialized features

## Usage

1. Create a Google Calendar client ID and API key 
2. Replace the variables in `index.html`
3. In the console
    - `npm install http-server`
    - `npx http-server -p 8000`
4. Navigate to `http://localhost:8000`
